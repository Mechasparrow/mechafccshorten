```https://mechafccshorten.glitch.me/new/https://www.google.com```
```https://mechafccshorten.glitch.me/new/http://foo.com:80```

```{ "original_url":"http://foo.com:80", "short_url":"https://mechafccshorten.glitch.me/8170" }```
